Estudando HTML

Nossos estudos começaram falando a respeito de um item muito sério que é a organização. Isso porque a linguagem HTML é responsável apenas pela marcação dos elementos na janela do navegador.

Parágrafos

Para colocarmos parágrafos, usamos os elementos "p" e "/p". Para fazer uma guebra de linha, mantendo o mesmo parágrafo, usamos o elemento "br". Temos ainda parágrafos especiais, denominadaos títulos, cabeçalhos ou headings em que são usados elementos "h1" a "h6".

Imagens

Uma imagem aparece na janela do navegador pelo uso do elemento "img".
